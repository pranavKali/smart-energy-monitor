# Smart Energy Monitor Project Plan

## Goal

Build a smart energy monitoring system using the ADE7880 and ESP32 that can:

- Measure voltage
- Measure current
- Calculate power
- Track energy usage
- Send data over Wi-Fi
- Display data on a phone or dashboard

## Learning Objectives

- Understand the ADE7880
- Learn ESP32 basics
- Learn SPI communication
- Learn embedded programming
- Learn PCB design
- Learn IoT dashboards

## Development Phases

1. Learn the hardware
2. Learn ADE7880 communication
3. Build firmware
4. Read energy data
5. Send data over Wi-Fi
6. Build dashboard
7. Create final prototype
