# ESP32 Selection

## Selected Board

ESP32 DevKit V1

## Why This Board?

- Very common
- Inexpensive
- Built-in Wi-Fi
- Built-in USB programming
- Large community support
- Lots of tutorials available

## Project Uses

The ESP32 will:

- Communicate with the ADE7880 using SPI
- Read energy measurements
- Connect to Wi-Fi
- Send data to a dashboard

## Important Specifications

- Dual-core processor
- 240 MHz clock speed
- Wi-Fi support
- Bluetooth support
- Multiple SPI peripherals
- 3.3V logic
