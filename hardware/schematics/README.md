# Hardware Schematics

## ADE7880 Analog Front End

### Description

This schematic contains the analog signal conditioning circuitry for the ADE7880.

Main functions:

- Voltage sensing
- Current transformer interface
- Input filtering
- ADE7880 analog inputs
- Clock circuitry
- Power decoupling
- MCU/isolator interface

### Status


Firmware will communicate with the ADE7880 through the SPI interface shown in the schematic.
